chartType parameter

	var obj = new Chart( );
        var chartType = obj.chartType.Name;

	- 1st parameter of drawChart( ) fn.

	- *required parameter

	- type is string

	- supported charts : line, bar (2d & 3d), pie (2d & 3d), 
	  column (2d & 3d) 

	- supported chart in multi series charts : line, bar, 
	  column ( all of these will be in 2d )

data parameter

	var data = [ ]; 

	- 2nd parameter of drawChart( ) fn.

	- *required parameter 

	- type is Array 

	- accepts obj. consists of lables and values etc.

options parameter

	var options = { }; 

	- 3rd parameter of drawChart( ) fn. 

	- *required parameter 

	- type is Object 

	- consists of many keys for customizing chart.

	- renderAt key decides on which html elemnt 
	  chart is to be drawn & this key is required

	-> for detailed description of all parameters 
	   please refer to fusionCharts/myjs/main.js location
